# `leo new`

新しいプロジェクトを作成するには次を実行します。
```bash
leo new <NAME>
```

プロジェクト名は `snake_case`（小文字と数字をアンダースコアで区切った形式）である必要があります。このコマンドは指定した名前のディレクトリを作成し、その中に初期構成を生成します。

詳細は [Project Layout](./../language/01_layout.md) を参照してください。
