## Leo Document (日本語)

Aleo、Leo プログラミング言語、およびその他のツールや概念に関するドキュメントソースファイルの[公開リポジトリ](https://github.com/ProvableHQ/leo-docs-source)の内容を日本語にしたものです。
その他のリソースはこちら：

- [📡 SnarkOS](http://snarkos.org/) - プライベートアプリケーション用の分散型オペレーティングシステム
- [⚙️ SnarkVM](https://snarkvm.org/) - ゼロ知識証明用のバーチャルマシン
- [🦁 Leo](https://leo-lang.org/) - ゼロ知識証明用のプログラミング言語
- [🛝 Playground](http://play.leo-lang.org) - すぐに ZK 開発とテストを始めるための Leo コンパイラへのブラウザインターフェース
- [🧰 SDK](https://provable.tools/) - Aleo 用のソフトウェア開発キット(SDK)
- [🔭 Explorer](https://explorer.provable.com) - Aleo ネットワーク用のブロックエクスプローラー

Aleo の日本コミュニティの[X アカウント](https://x.com/Aleo_Japan)をフォローして、わからないことがあれば質問してみましょう！
